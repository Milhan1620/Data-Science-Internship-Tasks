# Iris Dataset Analysis

## Objective

The objective of this project is to explore and visualize the Iris dataset using Python, Pandas, Matplotlib, and Seaborn.

## Approach

* Loaded the Iris dataset using Pandas and Seaborn.
* Inspected the dataset using shape, columns, and head functions.
* Checked and cleaned missing values.
* Performed Exploratory Data Analysis (EDA).
* Created Scatter Plot, Histogram, and Box Plot.
* Trained a Decision Tree Classification model.
* Evaluated the model using Accuracy Score and Confusion Matrix.

## Results and Insights

* The dataset contains three species of Iris flowers.
* Petal length and petal width are highly useful for distinguishing species.
* Visualizations helped identify feature distributions and relationships.
* The Decision Tree model achieved high classification accuracy.

* Pandas
* Matplotlib
* Seaborn
